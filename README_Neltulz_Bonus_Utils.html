Features

  • Delete Unselected Objects (and simultaneously remove orphaned collections)
  • Select Contiguous Edges - Allows you to grow a selection of edges forward, backwards, or in both directions based on an angle threshold.
  • Modifier Tools Panel & Pie Menu - Show/Hide modifiers, apply/remove modifiers, add modifiers, and open the modifiers sidebar easily.

Github

Install

  1. Download the Neltulz - Bonus Utils zip file anywhere on your computer.
  2. Run Blender.
  3. Go to Edit > Preferences.
  4. In the left sidebar, go to Add-ons.
  5. At the top right, click Install.
  6. Browse for the Neltulz - Bonus Utils zip file and click Install Add-on from File
    Alternatively, you may double click the zip file to install it.
  7. Next to Neltulz - Bonus Utils, enable the checkbox.
  8. (OPTIONAL) If you are manually saving preferences (instead of automatically), at the lower left of the preferences window, click Save Preferences and close the preferences window.

How to Use

  1. Open the sidebar in the 3d Viewport View > Sidebar
  2. Go to the Neltulz category and expand the Bonus Utils panel to access all of the bonus utilites.
  3. Press CTRL+SHIFT+Q to access the modifier tools pie menu.
Hotkeys
Note: Hotkey works in the 3D Viewport.
Hotkey Result Command Properties
CTRL+SHIFT+Q Open the "Modifier Tools" Pie Menu ntzbnsutls.modifiertoolspie None
ALT+Q Open the "All-in-One Tools" Popup Menu ntzbnsutls.aiotoolsettings None

Changelog

  • 2020-01-31
    • Added 3D Cursor options to Adjust, Apply, or Rotate
    • Added Snap Cursor+ feature which allows to snap cursor to selected object, face, edge, or vertice with orientation.
  • 2020-01-26
    • Added Normal Extrude+ feature, which functions similarly to Offset Faces but on a Mesh Face selection level
    • Added an All-in-one Tool Settings popup which contains most tool related options in a convenient popup.
    • Renamed Offset Faces to Offset All Faces
  • 2020-01-22
    • Added (Work in progress) Subdivide+ feature, which automatically selects newly added edges/vertices after subdividing.
    • Added Offset Faces feature, which uses a solidify modifier to offset faces by a specified amount.
